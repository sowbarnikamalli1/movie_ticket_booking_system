Movie Ticket Booking System

About the Project
This is a full-stack Movie Ticket Booking System that allows users to browse movies, view available shows, and book tickets online. The system is divided into two main parts: a frontend built using web technologies and a backend built using Java.

The project is designed to simulate a real-world movie ticket booking platform with basic features such as movie listing, seat selection, and booking management.

Features
- View list of available movies
- Check show timings
- Book movie tickets
- Seat selection system
- Backend API for handling bookings
- Simple and user-friendly interface

Tech Stack

Installation and Setup

Frontend Setup
1. Navigate to the frontend folder
2. Open index.html in a browser

Backend Setup
1. Open backend project in any Java IDE (Eclipse / IntelliJ / VS Code)
2. Configure server if required
3. Run the backend application

How to Use
1. Start the backend server
2. Open the frontend in browser
3. Browse movies and select a show
4. Choose seats and book tickets
